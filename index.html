Student Innovation Project (SIP)

Project Title

Cognitive Load-Aware Task and Learning Planner

Innovation Claim

This project proposes a cognitive load-aware task and learning planner that prioritizes tasks based on mental effort and complexity rather than task quantity or deadlines alone. By integrating cognitive load principles into planning, the system aims to reduce cognitive overload and support sustainable productivity.

Project Description

The project focuses on designing a functional digital planner prototype that allows users to input tasks, assign cognitive effort levels, and receive system-generated task prioritization. The scope includes interface design, task categorization logic, and usability testing. Advanced AI automation is intentionally excluded to maintain feasibility within the SIP timeline.

Proof of Concept / Early-Stage Prototype

The proof of concept includes low-fidelity wireframes and a system flow diagram demonstrating task input, cognitive effort tagging, and system-generated task prioritization.
